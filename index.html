SasView for Small Angle Scattering Analysis
A SAS Community Project launched from the NSF DANSE effort

SasView is a Small Angle Scattering Analysis Software Package developed as part of the NSF DANSE project, now managed by an international collaboration of facilities. Feedback and contributions are welcome and encouraged.

This web page is only a stub at the moment as we are preparing the first ever SasView release. This page should be fully active by the final release date which currently is Oct 19, 2012. In the meantime the predecessor SansView can be downloaded here.
