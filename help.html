How to get help

Much is documented in the help documentation. In particular, descriptions of each model, the equations used and appropriate references. This should always be the first place to go with questions. Also note that the bottom information bar on SasView often has useful information. Other available sources are

  • A tutorial (in pdf format)
  • The Model functions description (a pdf download of the model help file)
  • Visit the Help Archives here
  • You may contact us at: help@sasview.org to report bugs or for help using the software
  • Visit our tickets list To view reported bugs, things being worked on or possible future ehancements
  • For the adventurous, many new features and bug fixes are available in the latest builds (done at least daily) and may have what you are looking for
    • Official Windows and Ubuntu builds
    • Official Snowleopoard builds
