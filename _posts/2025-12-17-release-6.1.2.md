[Version 6.1.2](https://github.com/SasView/sasview/releases/tag/v6.1.2) was
released on Dec 17, 2025!

This is a minor bug fix release of SasView. It is built with Sasmodels v1.0.12,
Sasdata v0.11.0, and Bumps v1.0.2

**Bug Fixes**

* Fix for plugin models not being found 
* Fix for Wedge and Sector Slicers in Q using the same plot
* Replaces 'magic numbers' with enum
* Adds plotUpdateSignal to BoxSlicer
* Fix for Chi2 calculation in Inversion perspective
* Invariant bugfixes and 'TODO' tasks
* Removes redundant param
* Invariant reformat
* Adds module level logging to all files
* Adds function for close

**Documentation Fixes**

* Various fixes to try to get sasview sphinx docs to build cleanly.
* Include API docs from sasmodels in sasview sphinx output
* Update plugin model location in docs

**Python Cleanup and Code Formatting (Linting) Changes**

* Converts type comments to type hints
* Explicitly specifies latest version of ruff in CI
* Adds local pre-commit hook for ruff linting auto fixes

**Infrastructure Changes**

* Add dependabot for github actions
* Limit windows signing action
* Install innosetup in windows installer CI runs
* Bump actions/download-artifact from 4 to 5
* Bump dawidd6/action-download-artifact from 6 to 11
* Bump actions/checkout from 4 to 5
* Bump actions/setup-python from 5 to 6
* Bump digicert/ssm-code-signing from 1.0.1 to 1.1.1
* move qt gui build to cli.py and remove run.py
* Change remaining mentions of appdirs to platformdirs
* Replace macos-13 with macos-15-intel from macOS build
* Replace direct AUSAXS library integration with the pyAUSAXS wrapper

For more details, please see the Release Notes which are now available from the program Help or online.

**New Models**

No new models have been added to SasView or the [Model Marketplace](https://marketplace.sasview.org/) 
since v6.1.1 was released.
