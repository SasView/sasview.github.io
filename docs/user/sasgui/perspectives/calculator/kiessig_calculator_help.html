Kiessig Thickness Calculator Tool

Description

This tool estimates real space dimensions from the position or spacing of features in recipricol space. In particular a particle of size \(d\) will give rise to Bragg peaks with spacing \(\Delta q\) according to the relation

\[d = 2\pi / \Delta q\]

Similarly, the spacing between the peaks in Kiessig fringes in reflectometry data arise from layers of thickness \(d\).
