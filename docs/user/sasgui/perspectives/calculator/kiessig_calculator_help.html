Kiessig Thickness Calculator Tool

Description

This tool is approximately estimates the thickness of a layer or the diameter of particles from the position of the Kiessig fringe/Bragg peak in NR/SAS data using the relation

(thickness or size) = 2 * π / (fringe_width or peak position)

Using the tool

To get a rough thickness or particle size, simply type the fringe or peak position (in units of 1/Å) and click on the Compute button.
